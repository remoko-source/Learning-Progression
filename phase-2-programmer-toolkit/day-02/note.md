# Phase 2 - Day 2: Git via Terminal

## Yang Dipelajari

- `pwd` — cek posisi folder sekarang
- `ls` — lihat isi folder
- `cd` — pindah folder
- `git status` — cek kondisi repo
- `git add .` — staging semua perubahan
- `git commit -m "pesan"` — simpan snapshot
- `git push` — kirim ke GitHub
- `git remote set-url` — ganti URL remote
- `git log` — lihat riwayat commit lengkap
- `git log --oneline` — lihat riwayat commit ringkas

## Catatan Penting

- `-` vs `--` : tergantung command, bukan artinya universal
- `-m` di Python = module, di Git = message
- `--oneline` tidak punya shortcut
